# Noah Educa — Fluxos UX Padrão

## Fluxo 1 — Onboarding + Perfil (IA)
1. Boas-vindas → CTA: Começar
2. Coleta de perfil (renda, cidade, FGTS, entrada) → CTA: Gerar trilha
3. Trilha personalizada → CTA: Iniciar módulo 1

## Fluxo 2 — Simulador de poder de compra
Entrada + FGTS + renda → faixa de preço e parcela estimada
CTA: Salvar simulação | Sec: Comparar cenários

## Fluxo 3 — Checklist da jornada
Pesquisa → visita → negociação → contrato → financiamento → cartório → pós-compra
CTA: Marcar como concluído
